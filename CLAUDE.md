## 臉書長文規則

### 絕對禁用詞

| 禁用詞 | 替代用法 |
|--------|----------|
| 紅旗 | 警訊、疑點、怪的地方、不對勁、問題點 |
| 話術 | 說話的方法 |
| 底氣 | 自信 |
| 框架 | （直接不用，改換別的說法） |

### 用語規範

- 「學員」一律改為「諮詢者」
- 名人引言前面加 🗣️ emoji
- 對話敘述省略主詞（例：「問吃什麼」不要寫「問你吃什麼」）

### 格式規則

- 每行文字之間要有一行只含零寬空格（`​`）的空行
- 不使用粗體
- 結尾 hashtags 各自獨立一行：

```
#派特
#情感架構師
#聊天
#脫單
#教戀愛的祖師爺
#自我成長
```

## .docx 產出規則

1. 預設只給 Word 檔，**不附帶 PDF**（除非派特特別要求）
2. 字型四個 key 都要設：`{ ascii, cs, eastAsia, hAnsi }` 全部用 `"Arial Unicode MS"`
3. 空行段落用 `​`（零寬空格），TextRun **不要**設 font（連 `font: {}` 都不要）
4. 表頭陣列用 `" "`（單一空格）不要用 `""`，避免產生 `<0/>` 無效 XML
5. 內容大改時直接重生整支腳本，不要硬改原腳本
6. 解包資料夾每個版本獨立，避免污染
