## 1. 目標受眾分析

### 核心受眾：台灣男性，25–42 歲

這批人不是來「學搭訕」的。他們在感情裡遇到某種具體的卡關，開始在 Google 上搜尋。

根據 2026 戰略與關鍵詞分析，他們有三種進站路徑：

| 入口類型 | 具體樣貌 | 他們搜什麼 |
|---|---|---|
| **行動入口** | 想主動追，但不知道怎麼開口 | 怎麼認識女生、主動開口、怕被拒絕 |
| **痛點入口** | 已在關係裡，卡在情緒漩渦 | 患得患失、暈船、曖昧期怎麼辦 |
| **自我察覺入口** | 開始懷疑問題在自己身上 | 焦慮型依附、討好型人格、感情沒安全感 |

### 這三類人的共同底層需求

> **他們想要的不是技巧，是穩定感。**

他們不知道自己的問題叫「情感 OS 過舊」，但他們知道自己在感情裡容易失控、患得患失、靠對方的反應決定自己的狀態。這是網站訊息設計的核心鉤點。

### 次要受眾

- **女性，28–38 歲**：用「焦慮型依附」「討好型人格」「沒安全感」這類自我覺察詞搜尋的女性，佔比約 20-30%，可透過文章自然導入，但產品主力仍以男性為主。

## 2. 核心關鍵字布局

### 2.1 主關鍵字（網站要搶的賽道）

| 關鍵字 | 搜尋量推估 | 競爭難度 | 備註 |
|---|---|---|---|
| 患得患失 | 高 | 中（無強勢個人品牌主導） | 最大痛點池，主攻 |
| 焦慮型依附 | 中高（持續成長） | 低中 | 心理學詞彙流量上升中 |
| 暈船 | 高 | 中（需過濾「暈車藥」流量） | 標題加「感情」限縮意圖 |
| 主動認識女生 | 中 | 低 | 2026 男性行動入口，差異化高 |
| 討好型人格 | 高 | 中 | 自我察覺入口 |
| 怕被拒絕 | 中 | 低 | 搭配行動入口文章 |

### 2.2 長尾關鍵字（文章標題優先用這些）

**患得患失系列：**
- 患得患失 怎麼辦 改善
- 喜歡一個人但很不安怎麼辦
- 戀愛中 感覺對方不在乎我 怎麼辦
- 患得患失的根本原因

**主動行動力系列：**
- 怎麼主動認識女生 不尷尬
- 開口說話很緊張 感情
- 害怕被拒絕 怎麼讓自己開口

**暈船 / 曖昧系列：**
- 暈船一個人 怎麼讓自己好過
- 曖昧期對方不主動 我該怎麼辦
- 喜歡一個人但不確定對方是否喜歡我

**自我察覺系列：**
- 焦慮型依附 戀愛 怎麼辦
- 討好型人格 感情 怎麼改
- 感情沒安全感 是我的問題嗎
- 為什麼我在感情裡很沒自信

### 2.3 絕對不碰的關鍵字（docx 分析結論）

| 避開詞 | 原因 |
|---|---|
| 脫單（當主軸） | 80% 搜尋量是「全國脫單能力考試」心測娛樂，真實需求極低 |
| 情緒勒索 | PUA / gaslighting 流量嚴重污染，TA 完全不同 |
| 內耗（當主類） | 周慕姿個人品牌已強勢獨佔，正面競爭無意義 |
| 搭訕（當主類） | 馬克 / Edward 等搭訕教練品牌已佔領，且與情感 OS 定位衝突 |

> **替代策略**：「停止感情內耗」可當副標題使用；「主動」「行動力」取代「搭訕」。

## 3. 網站架構

### 3.1 整體頁面地圖

```
五四愛情學苑官網
│
├── 首頁（定位 + 分流）
│
├── 關於派特（信任建立）
│
├── 情感 OS 3.0（核心方法論頁）
│
├── 服務
│   ├── 一對一諮詢
│   ├── 月度訓練課程
│   └── 關懷之夜直播
│
├── 文章中心（SEO 主引擎）
│   ├── 患得患失與安全感
│   ├── 主動行動力
│   ├── 暈船與曖昧
│   └── 焦慮型依附與自我成長
│
└── 預約 / 聯絡
```

### 3.2 頁面關係與流量邏輯

```
Google 搜尋
    ↓
文章（痛點關鍵字）
    ↓
內文 CTA 連結至「情感 OS 3.0 方法論頁」或「服務頁」
    ↓
預約諮詢 / 加入關懷之夜 / 報名課程
```

文章是流量入口，不是最終目的地。每篇文章結尾都要有清楚的「下一步」。

## 4. 每頁的訊息層級

### 4.1 首頁

**目標：讓不同入口的訪客都能在 5 秒內感覺「這是說我的」**

| 區塊 | 內容設計 | 訊息重點 |
|---|---|---|
| Hero | H1 主標題 | 直打痛點，不談技巧 |
| 副標題 | 一句說明「你的問題是什麼」 | 框定受眾，讓對的人留下 |
| 社會證明 | 諮詢者案例 / 年資數字 | 「19 年 × X 位諮詢者」 |
| 分流模組 | 三個入口 CTA | 讓不同階段的人找到適合自己的路 |
| 第二屏 | 情感 OS 核心概念預告 | 建立「這不是一般戀愛教練」的差異感 |
| 底部 CTA | 雙 CTA 設計 | 主：預約諮詢；次：加入關懷之夜 |

**建議首頁 H1 選項（擇一）：**
- 「你不是不會戀愛，你的情感 OS 需要更新。」
- 「感情裡的患得患失，不是態度問題，是系統問題。」
- 「19 年，我見過的感情問題只有一個根源。」

**三入口分流 CTA：**
- 「我想主動，但不知道怎麼開口」→ 行動力文章
- 「我在關係裡很不安，患得患失」→ 安全感文章 or 情感 OS 頁
- 「我懷疑問題出在我自己」→ 預約一對一諮詢

---

### 4.2 關於派特頁

**目標：建立信任，讓訪客相信派特的 19 年是真的**

訊息順序：
1. 一句話：派特是誰、幫什麼樣的人
2. 為什麼懂這件事（背景故事，不只是資歷清單）
3. 核心主張：情感 OS 3.0 的哲學（「價值感來自內在，不是別人的評價」）
4. 媒體曝光 / 授課紀錄 / 諮詢個案數字
5. CTA：預約諮詢 or 進一步了解方法論

---

### 4.3 情感 OS 3.0 方法論頁

**目標：這是網站的智識核心，讓訪客從「懷疑」變成「想要」**

訊息層級：
1. 痛點確認：「為什麼你在感情裡一直卡在同一個地方」
2. 問題根源：情感 OS 過舊的概念解釋（用白話，不用術語）
3. 方法論介紹：OS 1.0 → 2.0 → 3.0 的差異（圖表或對照表效果最好）
4. 真實案例：某類型諮詢者的前後對比（匿名化處理）
5. CTA：「想知道你現在是哪個版本？」→ 預約診斷

---

### 4.4 一對一諮詢服務頁

**目標：讓已經準備好付費的人完成預約**

訊息層級：
1. 這是給誰的（明確說出適合 / 不適合的人）
2. 諮詢的流程（幾次、怎麼進行）
3. 諮詢後會得到什麼（具體成果，不是模糊的「更好的自己」）
4. 費用 / 方案說明
5. 常見問題 FAQ
6. 預約 CTA（唯一，不分散注意力）

---

### 4.5 文章頁（SEO 主力）

每篇文章遵循以下訊息結構（對應 docx 語言層級對照表）：

| 段落 | 語言類型 | 目的 |
|---|---|---|
| 標題 + 開頭 | 大眾感受詞（患得患失、暈船、怕被拒絕） | 讓 Google 找得到、讓讀者點進來 |
| 問題解析 | 心理學標籤（焦慮型依附、價值感外包） | 建立派特的專業感 |
| 改變路徑 | 自我成長語言（重建自信、停止討好） | 讓讀者感覺有希望 |
| 文末 CTA | 情感 OS 3.0 連結 | 從文章導向方法論或服務 |

**文末 CTA 三段式設計：**
1. 行動 CTA（馬上可做）：「如果你還在這個狀態，先看這篇：[相關文章]」
2. 方法論 CTA：「這個問題的根源我在這裡有更完整的說明：[情感 OS 3.0 頁]」
3. 諮詢 CTA：「如果你想知道自己的情況，可以預約一次診斷：[預約頁]」

## 6. SEO 基礎設定建議

### 6.1 網域建議

優先考慮選項（可用度需另行確認）：
- `pat54.com` — 簡短、記憶點高、結合派特 + 五四
- `pat-love.tw` — 直觀、台灣域名有本地信任感
- `54love.tw` — 品牌感強，搜尋品牌詞時更清楚

**建議用 `.tw` 或 `.com.tw`** — 台灣 Google 搜尋結果對本地域名有輕微偏好。

---

### 6.2 結構化資料（Schema Markup）

| Schema 類型 | 套用頁面 | 目的 |
|---|---|---|
| `Person` | 關於派特頁 | 讓 Google 知道派特是真實的人物品牌 |
| `Article` | 每篇文章 | 增加文章在搜尋結果的版位（可能帶出發佈日、作者） |
| `FAQPage` | 服務頁、方法論頁 | 搶佔 FAQ 富摘要版位，增加搜尋結果面積 |
| `LocalBusiness` | 首頁 / 聯絡頁 | 若有實體課程地點，強化本地搜尋 |

---

### 6.3 技術效能目標

| 指標 | 目標值 | 說明 |
|---|---|---|
| LCP（最大內容渲染） | < 2.5 秒 | 首屏主圖 / 標題載入速度 |
| CLS（版面位移） | < 0.1 | 避免圖片、字型載入後跑版 |
| 行動版體驗 | 滿分 | 台灣行動搜尋佔比高，必須優先 |
| 圖片格式 | WebP | 所有圖片轉 WebP，減少載入時間 |

---

### 6.4 內部連結策略

- 每篇文章至少連結到 1 篇相關文章 + 1 個服務頁 or 方法論頁
- 「情感 OS 3.0 方法論頁」是全站最重要的頁面，每篇文章都要有機會連到這裡
- 建立「文章群組（Content Cluster）」：每個主題類別有 1 篇「柱文章（Pillar Page）」+ 多篇支撐文章

**柱文章設計（每類一篇，2000 字以上）：**
- 患得患失完整指南（連結所有患得患失相關文章）
- 焦慮型依附完整指南
- 曖昧期完整指南
- 主動認識人完整指南

---

### 6.5 Meta 標籤規範

每頁 Title 格式：`[文章主題] | 派特 – 情感教練`
Description：80–120 字，包含主關鍵字 + 一個明確的行動誘因

## 7. 成功指標

### 7.1 流量指標（前 6 個月）

| 指標 | 第 3 個月目標 | 第 6 個月目標 |
|---|---|---|
| 月有機搜尋流量 | 500 人次 | 2,000 人次 |
| 主力關鍵字排名 | 前 10（至少 2 個） | 前 5（至少 3 個） |
| 平均頁面停留時間 | > 90 秒 | > 120 秒 |
| 跳出率 | < 75% | < 65% |

### 7.2 轉換指標

| 指標 | 說明 | 目標 |
|---|---|---|
| 諮詢預約率 | 造訪 → 填預約表單 | 1–2%（100 人進來 1-2 個預約） |
| 關懷之夜加入率 | 文章讀者 → 加入直播 | 3–5% |
| LINE / 社群連結點擊 | 文章底部社群連結 | 每月穩定成長 |

### 7.3 內容指標

| 指標 | 目標 |
|---|---|
| 開站時文章數 | 至少 5 篇（第一批全部完成） |
| 月更新頻率 | 2–4 篇 |
| 臉書長文轉 SEO 文章 | 每月至少 1 篇改寫 |

### 7.4 怎麼知道網站「有效」

最直接的判斷指標，按重要性排序：

1. **預約諮詢的人說「我是從網站找到你的」** — 這是最終目的
2. Google Search Console 顯示「患得患失」「焦慮型依附」等主力詞的排名進入前 10
3. 月有機流量連續 3 個月成長
4. 文章平均閱讀時間 > 2 分鐘（代表內容真的被讀完）
